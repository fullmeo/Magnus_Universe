## PARTIE 1: LA NOTE SENSIBLE DANS LA MUSIQUE (FONDATION)

### Qu'est-ce que la Note Sensible?

En musique, la **sensible** est la note qui crée une **tension intolérable** qui demande absolument une résolution.

```
Do (Ut) ─────────────────────────────────────────
                                        ↑
                                    Si (tension)
                                        ↓
                                        DOIT résoudre
                                        ↓
                                    Do (Ut) - RÉSOLUTION

C'est IMPOSSIBLE musicalement de rester sur Si.
La structure tonale FORCE la résolution.
```

### Le Paradoxe de la Sensible

**C'est une OBLIGATION qui vient DE LA STRUCTURE ELLE-MÊME**, pas d'une autorité externe.

```javascript
// La sensible ne demande pas "s'il te plaît, résous"
// La sensible EST UNE TENSION qui se résout
// C'est comme respiré - impossible de ne pas faire

// La musique génère sa propre NÉCESSITÉ
// C'est la liberté dans la structure
```
