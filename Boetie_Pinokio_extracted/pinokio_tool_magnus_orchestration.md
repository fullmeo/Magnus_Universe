## PARTIE 5: LE MÉCANISME D'ISOLATION DANS PINOKIO

### Sécurité par Isolation

Par défaut, toutes les applications Pinokio s'exécutent dans un emplacement isolé (~/pinokio/api). De plus, tous les fichiers binaires installés via les gestionnaires de paquets intégrés de Pinokio sont installés dans ~/pinokio/bin. Fondamentalement, tout ce que vous faites est stocké à l'intérieur de ~/pinokio.

```javascript
// PINOKIO: Isolation de l'environnement
~/.pinokio/
├─ api/
│  ├─ app1/ (isolated venv)
│  ├─ app2/ (isolated venv)
│  └─ app3/ (isolated venv)
└─ bin/ (isolated binaries)

// Chaque application est ISOLÉE
// Pas de pollution cross-app
// Chaque app a son propre scope
```
