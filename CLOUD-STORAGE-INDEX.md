## ğŸ“ Quick Reference

### One-Liner
```javascript
const magnus = new Magnus13Extended({ cloudStorage: true });
```

### Three-Liner
```javascript
import { Magnus13Extended } from './index.js';
const magnus = new Magnus13Extended({ cloudStorage: true });
await magnus.initialize();
```

### Full Setup
```javascript
import { Magnus13Extended } from './index.js';

const magnus = new Magnus13Extended({
  cloudStorage: true,
  autoBackup: true,
  backupInterval: 3600000,
  backupOnGeneration: true
});

await magnus.initialize();
```

### Key Methods
```javascript
await magnus.backupToCloud();       // Manual backup
await magnus.restoreFromCloud();    // Restore from cloud
await magnus.syncWithCloud();       // Sync with cloud
await magnus.getCloudStats();       // Get stats
```
