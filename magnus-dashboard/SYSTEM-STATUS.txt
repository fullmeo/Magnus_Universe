SUPPORT AND TROUBLESHOOTING
===============================================================================

Common Issues:

Port Already in Use:
  $ lsof -i :3000
  $ kill -9 <PID>
  Or use different port: DASHBOARD_PORT=3001 npm run infinity:full

Scanner Not Found:
  System automatically uses mock implementation
  Full scanner will load if C:\Users\diase\magnus-scanner-14 available

CloudZero Not Available:
  Cloud Sync is optional - system works without it
  Disable explicitly: ENABLE_CLOUD_SYNC=false npm run infinity

Kill Switch Triggered:
  Check console for reason, fix issue, and restart:
  npm run infinity
